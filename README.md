### 各科技巧
* Listening: 我會先跳過 Part1, 開始畫 Part2、3 的問題和答案的關鍵字。聽完每 part 的空檔, 不要檢查, 繼續往後畫重點。遇到配對題, 我會將配對答案的關鍵字寫在紙上(考場會提供), 在聽的時候可以專心在關鍵字。Part4 關鍵字可以拯救你, 當你漏聽時。    
註記: 正式考的 part4 感覺有加速 (約雅思哥 1.2 倍速)
    * 自我準備:
        1. BBC Learning English (1.25 ~ 1.5倍速)
        2. 寫雅思哥
        3. 背單字
    * 補習班: 沒有太大幫助, 還是需要大量真題去熟悉題型和寫題速度    

* Reading: 三篇是有難度之分, 對每篇要有時間規劃。題目通常是會照著內容順序, 我會先看題目然後掃關鍵字去找答案。
    * 自我準備:
        1. 寫雅思哥
    * 補習班: 同 Listening

* Writing: 速度!!! 練習雅思哥時, 要一次一起練習(Task1, Task2)並計時, 盡量將完成時間壓在50分鐘左右完成。Task2 先寫, 因為分數占比高。Task1 的考試內容蠻固定的, 時常練習可以寫很快很好。
    * 自我準備:
        1. Youtube: Lilie IELTS (推推)
        2. 寫雅思哥真題給 chatgpt 更正和打分
        3. 網路上提供的9分範例
    * 補習班: 最有幫助。因為雅思有固定文章架構, 老師會帶過每種題目並提供一些萬用詞 (像是百分比, 比較, 開頭等), 每個禮拜都會有一篇作業, 老師會批改並給建議。

* Speaking:  口說的考試時間很短, 可以建立自己有脈絡和架構的回覆 (用於Part1,3), 讓內容更豐富且多樣。不要經常重複詞彙和句子, 會讓考官覺得單字量不足。
    * 自我準備:
        1. 看小紅書或抖音上的範例
        2. 和 chatgpt, 雅思哥模擬雅思口說考試
    * 補習班:
        1. Pros: 分班制度之下同學程度差不多, 比較不會羞於開口, 也聽得懂對方的內容。
        2. Cons: 對話時不能即時糾正文法和發音錯誤, 即使是小班教學 (10人內) 會有和老師對話時間, 但可能在短期兩個多月進步不大。

### 心得
* 選擇機考和紙筆的關鍵 - Reading, Writing
    * 機考
        1. Reading 可以平行閱讀, 題目和文章佔螢幕各半, 可以很方便直接找答案和看關鍵字。
        2. Writing 可以複製貼上 ; 需要練習打字速度。
    * 紙筆
        1. Reading 畫重點比較快, 有些人習慣看紙本 ; 如果題目和文章在正反面需要一直翻來翻去。
        2. Writing 手寫通常比較快 ; 寫錯要塗塗改改, 字數要自己算。
* Listening, Reading: 應該是我們提升分數最快的考科, 所以二戰的重心都是在這兩科
* Listening: 一戰 part4 太快加緊張都沒聽到。二戰 part2,3 配對題和多選題很多也有單字不會拼, 但沒有一戰這麼慘烈。
* Reading: 一戰被聽力影響, 都讀不進去也沒讀完。二戰 Heading 還是覺得有點難。
* Writing: 一戰 Task1 差20字沒寫完, Task2 也沒寫很完整。二戰都有寫完, 但 Task1 寫的有點重複, Task2 可能有點篇題。兩次分數都是6.0分@@
* Speaking: 二戰比一戰不緊張也比較順, 但兩次分數都是6.0分@@    
=> 可以請一對一 Writing 和 Speaking
